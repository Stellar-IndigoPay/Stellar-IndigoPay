# Contributors

First and foremost: **thank you** to everyone who builds, documents, tests, designs, and
reviewes Stellar-IndigoPay. This file exists to recognize the people who make the project
better, one contribution at a time.

Stellar-IndigoPay is an open-source climate donation platform built on the Stellar network.
Every merged PR — whether it's a line of Rust, a React component, a docs fix, or a bug report —
earns a place here.

## How to be added

You do **not** need to edit this file by hand. Getting recognized is automatic once your
contribution lands:

1. Open a pull request against this repository.
2. Have it reviewed and **merged** by a maintainer.
3. You'll be added to the table below (role is derived from the kind of change you made).

Prefer the automated route? The project can use the
[@all-contributors](https://allcontributors.org/) bot — comment on an issue or PR with:

```
@all-contributors please add @your-username for <code|doc|design|test|ideas|review|bug>
```

A maintainer will trigger the update. Manual edits via PR are also welcome if the bot isn't
wired up yet.
